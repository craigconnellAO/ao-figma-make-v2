Notes

  • Naming. All groups/items carry data-aods="radio-button-group" | "radio-button" | "checkbox-group" | "checkbox" so prototype tooling can map back to @ao/components.
  • Selection theme. Always blue (action-secondary). Green stays reserved for the primary CTA in the same view.
  • Disabled items. Stay visible and explain why ("Unavailable") — don't hide them.
  • Inline rows. Use flex: 1 1 0 so columns stay equal width even with different content lengths.
  • Focus ring. 4px action-secondary-glow halo via :has(:focus-visible) on the row — never suppress it.
