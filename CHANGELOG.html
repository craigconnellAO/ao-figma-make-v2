v2.0 15 May 2026

The first shareable version. Restructured from scratch around three problems v1 couldn't solve.

What v1 got wrong

  1. AI drift. Frames pasted from Figma into Make drifted — wrong greens, invented variants, type hierarchy slipping.
  2. No confident dev path. No clear mapping back to @ao/components.
  3. The files were a mess. Provenance unclear, contents undocumented.

What changed in v2

  • Workflow: attach the kit, paste a short prompt. No more big system block every chat.
  • Authoritative reference files in kit/: tokens, typography, components, patterns, anti-patterns, DTCG tokens.
  • Anti-patterns are first-class. Refuse-and-replace list, hard constraint not guideline.
  • Dev handoff built in. Class-to-@ao/components translation table in handoff.md.
  • Examples are first-class. Rendered HTML for visual comparison.
  • Brand intent enforced. Token usage rules spelled out where Make sees them.
